# Requerimientos Historicos de ComunidadIFTS

## Alcance de este archivo

Este archivo conserva pedidos funcionales y backlog historico que antes estaban dispersos en markdown de raiz. No describe necesariamente el estado actual implementado.

## Tienda ComunidadIFTS

Se registro la idea de una tienda separada para productos relacionados con la comunidad, con Angular y PHP, login integrado con ComunidadIFTS y componentes reutilizables. Ese alcance evoluciono luego al proyecto independiente `ComunidadIFTS-Tienda`.

## Contacto

Se pidio un formulario de contacto inspirado en IFTS15 para recibir consultas desde la app.

## Perfil de alumno

Se pidio que el alumno vea:

- foto de perfil,
- datos personales,
- IFTS, carrera y ano,
- ofertas a las que se postulo,
- posibilidad de ocultar o cancelar postulaciones cuando la oferta deje de ser visible o ya no quiera verla.

## Perfil de AdministradorIFTS

Se pidio que la institucion vea:

- logo como foto de perfil,
- datos de contacto editables,
- cantidad de ofertas publicadas,
- cantidad total de postulantes,
- listado de ofertas con toggles para habilitar, deshabilitar o cancelar.

## Dashboard de AdministradorIFTS

Se pidio un panel con:

- titulo de oferta,
- fecha de publicacion,
- usuario postulante,
- apellido,
- nombre,
- descarga de CV,
- estadisticas de ofertas y postulantes.

## Bolsa de trabajo

Quedo pedido historicamente:

- solo instituciones registradas publican ofertas,
- solo alumnos registrados ven ofertas,
- las ofertas pasan por aprobacion de administrador de comunidad,
- postulacion con CV adjunto,
- notificaciones por mail a institucion y postulante,
- vista de gestion de ofertas con pendientes y publicadas.

## Nota

Para saber que parte de estos pedidos se encuentra realmente implementada, usar `docs/estado-actual.md`, `docs/historial-tecnico.md` y los QA documentados.
